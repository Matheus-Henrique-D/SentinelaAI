# 🚀 Instruções Rápidas - Sentinela Verde

## ⚡ **Início Rápido**

### **1. Execute a Interface**
```bash
cd SentinelaAI
python demo_simple.py
```

### **2. Teste a Interatividade**
- Clique no botão **"Gerar Novos Dados"** (canto superior direito)
- Observe como os valores mudam
- Veja a qualidade do ar mudar (Excelente → Bom → Moderado → Ruim)

## 🔄 **Funcionalidades do Botão**

### **Valores Gerados Automaticamente:**
- **Concentração**: 50-800 ppm
- **PM2.5**: 5-60 µg/m³
- **PM10**: 10-120 µg/m³
- **Temperatura**: 15-35°C
- **Umidade**: 30-90%

### **Qualidade do Ar (Muda Automaticamente):**
- 🟢 **Excelente**: Valores baixos
- 🟡 **Bom**: Valores moderados
- 🟠 **Moderado**: Valores elevados
- 🔴 **Ruim**: Valores altos

## 🛠️ **Se algo não funcionar:**

### **Problema**: "Módulo não encontrado"
**Solução**: 
```bash
pip install -r requirements.txt
```

### **Problema**: "Interface não abre"
**Solução**: 
```bash
python --version  # Verifique se Python está instalado
```

### **Problema**: "Erro de sintaxe"
**Solução**: Use a versão simplificada
```bash
python demo_simple.py
```
